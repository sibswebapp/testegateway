    Processo de Integração Gateway

    Nesta página pode consultar os detalhes da sua transação MB WAY. É obrigatório indicar o prazo máximo de 4 minutos para a sua conclusão, podendo fazê-lo através de um contador ou, em alternativa, através de uma mensagem informativa que comunique ao cliente este limite de tempo.

    Para obter estes dados necessita de fazer uma chamada API com o método:

    GET https://spg.qly.site1.sibs.pt/api/v2/payments/{transactionID}/status


    Mas depois necessita de utilizar um webhook para saber o estado real desta transação (o webhook é de caráter obrigatório)
