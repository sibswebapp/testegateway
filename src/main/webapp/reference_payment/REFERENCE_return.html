Nesta página pode visualizar os detalhes da referência bancária, como a entidade, referência, montante e data de expiração.

Para obter estes dados necessita de fazer uma chamada API com o método:

GET https://spg.qly.site1.sibs.pt/api/v2/payments/{transactionID}/status


Mas depois necessita de utilizar um webhook para saber o estado real desta transação (o webhook é de caráter obrigatório)
