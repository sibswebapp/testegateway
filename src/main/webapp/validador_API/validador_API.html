Exemplo de Body – CIT
Nota: MITs podem ser UCOF quando:
  • merchantInitiatedTransaction.type = "UCOF"
  • paymentType = "AUTH"
  • amount = 0
Ou RCRR quando:
  • merchantInitiatedTransaction.type = "RCRR"
  • paymentType = "PURS"
  • amount > 0
