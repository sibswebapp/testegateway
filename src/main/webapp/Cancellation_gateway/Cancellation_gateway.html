Processo de Cancelamento

Nesta página pode efetuar um cancelamento de uma transação. Para isso, deve preencher o Transaction ID, o valor a devolver e uma breve descrição.

Após submeter o cancelamento, será enviada uma requisição para a API:

POST https://spg.qly.site1.sibs.pt/api/v2/payments/{transactionID}/cancellation


O resultado da operação será apresentado no painel de Debug abaixo. Se o cancelamento for realizado com sucesso, será indicado com uma mensagem de confirmação.

Para informações detalhadas, consulte a documentação oficial:
